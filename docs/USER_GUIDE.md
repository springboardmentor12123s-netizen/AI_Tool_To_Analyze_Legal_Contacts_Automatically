# ClauseAI User Guide

Welcome to ClauseAI! This guide will help you navigate and use the Multi-Document Contract Analysis Dashboard effectively.

## 1. Uploading Documents
- On the left sidebar, locate the **Upload Documents** section.
- Click "Browse files" to select one or multiple contracts in `.pdf`, `.docx`, or `.txt` format.
- You can drag and drop files directly into the upload area.

## 2. Configuring the Report
Below the file uploader, you can configure how the AI should generate your report:
- **Tone**: Choose from `formal`, `concise`, or `executive`.
- **Structure Sections**: Select which sections you'd like the report to include (e.g., Summary, Risks, Clauses, Recommendations).
- **Focus Areas**: Highlight specific risk domains such as Liability, Data Privacy, or Termination clauses for the AI to prioritize.

## 3. Running the Analysis
- Click the **Deep Extract & Analyze All** button.
- The dashboard will upload all documents securely and process them concurrently.
- The **Analysis Status** panel will update with the progress and show a confirmation mark (✓) when a document is fully analyzed.

## 4. Viewing Reports and Giving Feedback
- Click on any document in the **Document List** panel.
- The generated report will populate in the main view.
- Expand the **Raw Extracted JSON** section to see specific clause mappings.
- **Feedback**: Rate the analysis using the Thumbs Up/Down buttons to help improve the system's accuracy.

## 5. Chat with Document (RAG)
- Select a document from the Document List.
- Use the chat box at the bottom to ask the AI questions directly about the selected contract.
- The AI uses Retrieval-Augmented Generation (RAG) to fetch relevant clauses and answer your queries.
